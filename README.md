# Projetos Em Java

Este repositório contém 5 projetos desenvolvidos em Java, cada um explorando diferentes conceitos de programação e lógica. Perfeitos para iniciantes e para praticar habilidades em Java!

---

## Projetos sem POO

### CalculadoraScanner
**Descrição:** Uma calculadora interativa que realiza operações matemáticas básicas (soma, subtração, multiplicação e divisão). Utiliza a classe `Scanner` para capturar a entrada do usuário e exibir os resultados.
**Conceitos:** Entrada de dados, estrutura de decisão (if/switch), operadores aritméticos.

### ControleDeGastos
**Descrição:** Um sistema para gerenciar finanças pessoais, permitindo registrar e categorizar despesas e receitas. Pode incluir funcionalidades como cálculo de saldo, listagem de transações e alertas de orçamento.
**Conceitos:** Manipulação de listas/arrays, laços de repetição, lógica de cálculos financeiros.

### JogoAdivinhacao
**Descrição:** Um jogo clássico onde o computador gera um número aleatório e o usuário deve adivinhá-lo. O programa fornece dicas como "maior" ou "menor" a cada tentativa, contando o número de palpites até o acerto.
**Conceitos:** Geração de números aleatórios (Random), laços de repetição (while/do-while), estruturas condicionais.

### OrganizadorDeEstudos
**Descrição:** Uma ferramenta para auxiliar nos estudos, permitindo criar, organizar e gerenciar tarefas, matérias ou tópicos a serem revisados. Pode incluir funcionalidades como adicionar, remover e listar itens de estudo.
**Conceitos:** Gerenciamento de dados com coleções (ArrayList, HashMap), criação de menus interativos.

### SistemaEscolar
**Descrição:** Um sistema para gerenciar informações de uma escola, como cadastro de alunos, professores, disciplinas e notas. Pode permitir calcular médias, listar alunos por turma e gerar boletins simples.
**Conceitos:** Programação orientada a objetos (POO), classes e objetos, manipulação de dados estruturados.
